# Aamir Ali Program for KidsGuide
This interactive daily habit and behavior tracker is designed for Muslim families to motivate up to three children. It features a weight-based points system categorized across three priority zones (Top, Mid, and Low) with customizable child names. A native calendar selector lets parents track daily checklists, view real-time scores, and write personalized daily feedback. 
All compiled progress is securely stored on the local browser storage and can be backed up or restored using portable JSON files.

# Family Success & Discipline Hub 🏆

An interactive, kid-friendly daily behavior and habit tracker designed for Muslim families. This static web application supports up to three children, dynamically tracks daily/weekly/monthly points, includes target qualification thresholds, and offers parental feedback tracking.

## 🌟 Features
* **Multi-Child Tracking**: Manage daily goals for up to three children with customizable names.
* **Three-Tier Priority System**:
  * **Zone 1 (Top Priority - 3 Points)**: Daily prayers, obedience to parents, schoolwork, sleep discipline, Quran study, and polite speech.
  * **Zone 2 (Mid Priority - 2 Points)**: Cleanliness, respectful manners, Islamic phrases, and maintaining a quiet home.
  * **Zone 3 (Low Priority - 1 Point)**: Active play, screen time boundaries, science/history exploration, and rewards.
* **Trailing Leaderboards**: Automatically calculates cumulative points for the last 7 days (Weekly) and 30 days (Monthly) based on the selected calendar date.
* **Qualification Milestones**: Displays progress status against Weekly (100 pts) and Monthly (400 pts) target thresholds.
* **Parental Controls**: Password-protected reset button (Password: `test101`) to protect tracking records.
* **Data Portability**: Automatically saves all progress to the browser's local storage. Includes options to export/import the entire dataset as a portable `.json` file.

## 🚀 How to Run Locally
1. Download or clone this repository.
2. Double-click `index.html` to open it in any modern web browser.

## 🌐 Deploying to GitHub Pages
To make this app accessible on mobile phones and tablets for your family:
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages** inside your GitHub repository.
3. Under **Branch**, select `main` (or `master`) and `/root`, then click **Save**.
4. Within a few minutes, GitHub will provide a live URL (e.g., `https://aamirali310.github.io/kidsguide`).
